Ribbon SBC Upgrade Automation

Automates firmware upgrades for Ribbon SWe Edge, SBC 1000, and SBC 2000 devices.

Upgrade workflow:

  1. Login via web session (CGI)
  2. Scrape upgrade page form fields
  3. Pre-upgrade validation
  4. Config backup
  5. Firmware upload (swDownload_do.phpx)
  6. Wait for reboot (~5–10 min)
  7. Verify new version

Credentials are encrypted at rest with Fernet symmetric encryption.
